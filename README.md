# LogicFriday1

Re-Implementation of Logic Friday 1

The goal of this project is to provide an open-source alternative to the legacy logic synthesis software *Logic Friday 1*, which is not developed anymore and there are also no sources available.

## Attribution

This project contains also re-implementations of the *Sequential Interactive Synthesis* (*SIS*) and the *espresso* two-level boolean PLA logic minimizer.
