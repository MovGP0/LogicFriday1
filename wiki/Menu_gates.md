## **Gates Menu**

**Modify Gate Diagram**\
Open the gate diagram of the selected function for editing.

**Copy to Clipboard**\
Copy an image of the entire diagram to the clipboard.

**IC Package Info...**...\
Display a parts list of standard IC packages for the gate diagram.

**Trace Gate Logic**\
Display the logic state of gate inputs and outputs for selected input vectors. Logic states are indicated by color, with 1=red and 0=blue.

**Auto Redraw Diagram**\
Redraw a user-entered diagram with the internal drawing engine

**Zoom In**\
Zoom in on the output gate diagram.

**Zoom Out**\
Zoom out the output gate diagram.

**Zoom All**\
Zoom to include the entire output gate diagram in the display area.

**Delete Selected**\
Delete the currently selected objects in gate diagram entry.

**Submit**\
Submit a gate diagram when editing is complete.

**Cancel**\
Cancel gate diagram entry.
