## **Saving a function to a file**

To save a function to a file the first time:

1.  Select **File \| Save**. The "Save As" dialog will appear.
2.  Select the folder where you want to save the file.
3.  A file name is suggested. To change it, just type over it.
4.  Click **Save** to save the file. The extension .*lfcn* is appended automatically.

**Notes:**\

- To update a file you have already saved, just select **File \| Save**. The file will be updated silently.
- To save a function to a new location or with a different file name, select **File \| Save As**. The "Save As" dialog will appear.
