## **Equation Menu**

**Modify Equation**\
Open the equation of the selected function for editing.

**Sum of Products**\
Display the logic equation for the selected function in sum-of-products form.

**Product of Sums**\
Display the logic equation for the selected function in product-of-sums form.

**Factor**\
Factor the logic equation for the selected function.

**Undo**\
Equation entry: Undo the last operation.

**Redo**\
Equation entry: Redo an undone operation.

**Cut**\
Equation entry: Copy selected text to the clipboard and then delete it.

**Copy**\
Copy selected text to the clipboard.\

**Paste**\
Equation entry: Paste clipboard contents.

**Delete**\
Equation entry: Delete selected text.

**Select All**\
Select all text.

**Submit**\
Submit an equation when editing is complete.

**Cancel**\
Cancel equation entry.
