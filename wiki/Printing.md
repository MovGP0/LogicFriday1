## **Printing diagrams and equations**

To print a gate diagram or logic equation:

1.  Select **File \| Print**. The "Print" dialog will appear.
2.  In the "Print what" box select "Gate diagram" or "Equation text."
3.  If you want to set the margins or the page orientation click the **Page Setup...** button.
4.  Click **OK** to begin printing.

**Notes:**\

- When you print a gate diagram, Logic Friday will try to fit it onto a single sheet. If that cannot be done without making the diagram illegible, the diagram will be printed on two or more sheets in the style of a road atlas.
- The truth table cannot be printed directly, but you can [export it](Exporting_TT.md) to a CSV file, which can be opened in a spreadsheet application and printed.
