## **Opening a function file**

To open a function file:

1.  Select **File \| Open**. The "Open" dialog will appear.
2.  Select the folder where the file is located. Any files with the extension .*lfcn* will be displayed.
3.  Select the file you want to open.
4.  Click **Open** to open the file. The function will be added to the function list window and the program will switch to it automatically.
