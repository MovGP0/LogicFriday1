## **Entering equations**

To enter a function as one or more equations:

1.  Select **File \| New \| Logic Equation**.
2.  Type the equation(s) in the input window. It is not necessary to enter multiple equations on separate lines, but you can do so by using CTRL+ENTER to advance a line.
3.  Terminate each equation with a semicolon.
4.  To submit the equations(s) select **Equation \| Submi**t, or just press ENTER. To cancel equation entry select **Equation \| Cancel**, or press ESC.
