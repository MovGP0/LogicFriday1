## **Tracing gate logic in a diagram**

When you have a gate diagram for a function you can trace the logic flow through the network for any combination of input values.

To trace gate logic:

1.  Select **Gates \| Trace Gate Logic**. The diagram will be redrawn with color coding to indicate logic values for gates and connecting wires. Logic 1 is indicated by red, and logic 0 is indicated by blue. Current input variable values are shown on the buttons in the "Inputs" tool window. Initially the values are all logic 0.
2.  To change an input value just click the button for the variable.
3.  When you have set all the variable values, click **Trace** and the diagram will be redrawn.
