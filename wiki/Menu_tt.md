## **Truthtable Menu**

**Modify Truth Table**\
Open the truth table of the selected function for editing.

**Show True and Don't Care**\
Show only rows of the truth table for which an output is true or don't care. This is always the case when the function has been minimized and the [view](Menu_view.md) is set to minimized.

**Show All Rows**\
Show all rows of the truth table. Available only if the view is set to unminimized.

**True**\
Set selected outputs of a truth table open for editing to True.

**False**\
Set selected outputs of a truth table open for editing to False.

**Don't Care**\
Set selected outputs of a truth table open for editing to Don't Care (X).

**Invert**\
Invert selected outputs of a truth table open for editing (except Don't Care).

**Submit**\
Submit a truth table when editing is complete

**Cancel**\
Cancel truth table editing.
